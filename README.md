# str2hex

this script is for String to hex,

ForExample:
  
  g3nt13m4n@localhost:~/dev$ ./str2hex.py "/bin"
  [*]Inputed : /bin
  [-]HexEncoded: 2f62696e
  [+]Generated: 0x6e69622f
  [+]Generated: \x6e\x69\x62\x2f
  g3nt13m4n@localhost:~/dev$ 
  
  
  ./str2hex.py <word for encode>
  
  ./str2hex.py "/ls"
